Capabilities

Modular Agent Skills

Each skill follows the Agent Skills open standard for compatibility with any coding agent.

🎨
stitch-design

Unified entry point for Stitch design work. Handles prompt enhancement, design system synthesis, and high-fidelity screen generation via Stitch MCP.

$ npx skills add ... --skill stitch-design --global
🔄
stitch-loop

Generates a complete multi-page website from a single prompt using Stitch, with automated file organization and validation.

$ npx skills add ... --skill stitch-loop --global
⚛️
react:components

Converts Stitch screens to React component systems with automated validation and design token consistency.

$ npx skills add ... --skill react:components --global
🎬
remotion

Generates walkthrough videos from Stitch projects using Remotion with smooth transitions, zooming, and text overlays.

$ npx skills add ... --skill remotion --global
📐
design-md

Analyzes Stitch projects and generates comprehensive DESIGN.md files documenting design systems in semantic language optimized for generation.

$ npx skills add ... --skill design-md --global
✨
enhance-prompt

Transforms vague UI ideas into polished, Stitch-optimized prompts. Enhances specificity, adds UI/UX keywords, and structures output.

$ npx skills add ... --skill enhance-prompt --global
🧩
shadcn-ui

Expert guidance for integrating and building applications with shadcn/ui components — discover, install, customize, and optimize.

$ npx skills add ... --skill shadcn-ui --global
💎
taste-design

Semantic Design System Skill. Generates agent-friendly DESIGN.md with strict typography, calibrated color, and perpetual micro-motion.

$ npx skills add ... --skill taste-design --global
Workflow

Get Started in Seconds

Three simple steps to extend your AI coding agent with new capabilities.

01

Browse Skills

Use the CLI to explore all available skills in the repository. Each skill comes with documentation and examples.

02

Install Globally

One command installs any skill to all your active coding agents automatically — Cursor, Antigravity, Claude Code, and more.

03

Start Building

Skills are immediately active. Your AI agents gain new capabilities — from design generation to video walkthroughs.
